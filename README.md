# Mini_Project
# 🧊 SmartFridge AI

An intelligent food management system that combines **Multi-Agent AI** and **IoT sensors** to predict food expiry, reduce waste, and automate smart kitchen management.

---

## 📌 Project Overview

SmartFridge AI is a modern AI-powered smart refrigerator solution designed to monitor food freshness in real time. The system uses:

* 📡 IoT Sensors for environmental monitoring
* 🤖 Multi-Agent AI architecture
* 📸 Computer Vision for food recognition
* 🔮 Machine Learning for expiry prediction
* 🔔 Smart notifications and recipe suggestions

The goal is to minimize food waste while improving convenience and sustainability.

---

# ✨ Features

* 📸 **Visual Food Recognition**

  * Detects food items automatically using computer vision.

* 🌡️ **Environmental Monitoring**

  * Tracks temperature, humidity, and gas levels.

* 🔮 **AI Expiry Prediction**

  * Predicts actual freshness instead of relying only on expiry labels.

* 🔔 **Smart Notifications**

  * Alerts users about expiring items.

* 🍳 **Recipe Recommendations**

  * Suggests recipes using available ingredients.

* 🛒 **Shopping List Automation**

  * Creates smart shopping lists based on inventory.

* ☁️ **Cloud Synchronization**

  * Syncs data across devices securely.

---

# 🧠 Multi-Agent AI System

The project uses specialized AI agents working together.

| Agent                 | Responsibility                       |
| --------------------- | ------------------------------------ |
| 🎯 Coordinator Agent  | Manages communication between agents |
| 👁️ Vision Agent      | Detects food items and labels        |
| 📊 Prediction Agent   | Predicts expiry dates                |
| 🌡️ Environment Agent | Monitors fridge conditions           |
| 📣 Notification Agent | Sends alerts and reminders           |
| 🍽️ Recipe Agent      | Suggests meals and recipes           |
